# PoE Helper v2.1.0 Pro

這是一個專為《流亡黯道》(Path of Exile) 玩家設計的現代化開荒輔助工具，基於 **React + Electron** 打造。v2.1.0 帶來了更強大的自定義功能與介面優化，讓你的開荒體驗更加流暢。

## 🚀 v2.1.0 新增功能

- 🖼️ **多張速查表支援**：現在可以同時選擇多張圖片，並在檢視器中切換。
- ⌨️ **全自定義快捷鍵**：不再受限於 F8/F9/F10，你可以根據喜好設定任何按鍵。
- 🙈 **介面彈性隱藏**：可單獨隱藏「計時器」或「任務攻略」，大幅節省螢幕空間。
- 💾 **視窗狀態持久化**：自動記憶視窗的位置、大小與所有設定，重啟後無縫銜接。

## ✨ 核心功能亮點

- 📌 **頂層顯示**：永遠保持在遊戲視窗上方，支援調整透明度。
- ⏲️ **精準計時**：追蹤路徑進展，支援快速歸零。
- 📝 **任務攻略**：內建 ACT 1-10 攻略，支援勾選進度。
- 🔍 **Regex 管理**：快速複製常用的商人搜尋字串。
- 🌓 **視覺風格**：支援深色/淺色模式，質感 UI 設計。

## ⌨️ 快捷鍵說明 (可自定義)

預設快捷鍵如下，可於設定面板中修改：

| 功能 | 預設按鍵 | 說明 |
| :--- | :--- | :--- |
| **隱藏/顯示** | `F10` | 快速隱藏或恢復整個助手視窗 |
| **速查表** | `F9` | 開啟或切換多張圖片速查表 |
| **Regex** | `F8` | 開啟常用的 Regex 搜尋面板 |

## 📦 下載與安裝

你可以在 [Releases](../../releases) 頁面下載：
- **安裝版 (.exe)**：推薦使用，會建立桌面捷徑。
- **綠色免安裝版**：解壓縮後執行 `PoE 開荒助手 Pro.exe` 即可使用。

## 🛠️ 開發人員資訊

1. 確保已安裝 [Node.js](https://nodejs.org/)。
2. 進入子目錄：`cd poe_helper_web`
3. 安裝依賴：`npm install`
4. 開發模式：`npm run electron:dev`
5. 打包安裝檔：`npm run electron:build`

---
*本工具僅作為介面輔助，不包含任何自動化遊戲操作，符合遊戲規範。*


# PoE Helper v2.0.0 Pro

這是一個專為《流亡黯道》(Path of Exile) 玩家設計的現代化開荒輔助工具，基於 **React + Electron** 打造，提供極簡、高效且高度自訂的遊戲輔助體驗。

## ✨ 功能亮點

- 📌 **視窗置頂**：永遠顯示在遊戲畫面上方，支援透明度調整，不擋視線。
- ⏲️ **動態計時**：精確追蹤開荒用時，支援暫停與快速歸零。
- 📝 **自定義攻略**：內建編輯器，可自由修改或匯入專屬的 1-10 章開荒路線。
- 🔍 **Regex 複製管理 (F8)**：預設常用商人搜尋字串，點擊即複製，提升遊戲效率。
- 🖼️ **速查圖片檢視 (F9)**：支援 POB 天賦或速查表圖片，可自由縮放、平移。
- 🌓 **個性化 UI**：支援深色/淺色模式、三段字體大小調整。
- ⌨️ **快捷隱藏 (F10)**：一鍵隱藏/顯示視窗，應對突發戰鬥。

## ⌨️ 快捷鍵說明

| 快捷鍵 | 功能描述 |
| :--- | :--- |
| **F8** | 開啟 Regex 複製面板 |
| **F9** | 開啟/隱藏 速查表圖片 |
| **F10** | 切換整個助手視窗的顯示/隱藏 |


本專案使用 Vite 作為建構工具。

## 📦 技術棧

- **Frontend**: React, Tailwind CSS, Lucide React
- **Animation**: Framer Motion
- **Desktop**: Electron (IPC, Global Shortcuts)
- **Build**: Vite
